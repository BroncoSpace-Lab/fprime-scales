# scalesSvc::GdsCmdAuthMux

Component to handle fallback delegation to UART/TCP GDS Commands. TCP is on by default, if TCP goes down, command authority is prioritized to UART until TCP comes back, then the user most command CMD Auth to switch back to the TCP Gds

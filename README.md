# scales-fprime-lib

This repository contains all of the hardware-critical F Prime components required to use the SCALES hardware stack in an integrated system. hardware-critical components include hardware managers to monitor and control power, thermals, and core system capabilities.

This repository is meant to be implemented within an existing project as a Git Submodule and F Prime Library, to then be integrated within your exisiting F Prime deployment.

This repository also contains the cmake files required to run F Prime on the i.MX 8X. Instructions on how to use this are on our [docs page](https://scales-docs.readthedocs.io/en/latest/software_home/).
